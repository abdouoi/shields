Shields

This simple RESTful service provides redirects for site specific shields/badges and homepages. Currently supported sites are:
Site Path Description
Codecov.io codecov Static analysis code coverage
GitHub.com github Git repository
Plugins.gradle.org gradle_plugin Plugins for gradle
bintray.com jcenter Code artifacts
search.maven.org maven_central Code artifacts
tldrlegal.com tldrlegal Software license explanations
travis-ci.org travis-ci Continuous delivery
For any of these you can request a shield, which redirects to shields.io or the repos own shield url, or a homepage which will redirect to the specified homepage at the site.

For more info:

API Documentation

Homepage
URL /homepage/[codecov|github|gradle_plugin|jcenter|maven_central|tldrlegal|travis-ci]
Method GET
Params
Param Description Required By
group The group name. gradle_plugin, jcenter, maven_central
package The package name. codecov, github, gradle_plugin, jcenter, maven_central, tldrlegal, travis-ci
path The path to the repo, minus package name. codecov, github, jcenter, travis-ci
Example ./homepage/maven_central?group=com.github.nwillc&package=almost-functional
Success Redirects to the homepage on the site for the package specified.
Shield
URL /shield/[codecov|github|gradle_plugin|jcenter|maven_central|tldrlegal|travis-ci]
Method GET
Params
Param Description Required By
group The group name. maven_central
package The package name. codecov, jcenter, maven_central, tldrlegal, travis-ci
path The path to the repo, minus package name. codecov, jcenter, travis-ci
Example ./shield/maven_central?group=com.github.nwillc&package=almost-functional
Success Redirects to a shield indicating the site, and the latest version if it can be determined.

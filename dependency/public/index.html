Repo Redirect

This simple RESTful service to provide redirects for repository specific shields/badges and homepages. Currently supported repositories are:
  • codecov
  • Github
  • Gradle Plugins
  • JCenter
  • Maven Central
For any of these you can request a shield, which redirects to shields.io or the repos own shield url, or a homepage which will redirect to the specified homepage at the repository.

For more info:

API Documentation

Homepage
URL /homepage/[codecov|github|gradle_plugin|jcenter|maven_central]
Method GET
Params
Param Description Required By
group The group name. gradle_plugin, jcenter, maven_central
package The package name. codecov, github, gradle_plugin, jcenter, maven_central
path The path to the repo, minus package name. codecov, github, jcenter
Example http://repo-redirect.herokuapp.com/homepage/maven_central?group=com.github.nwillc&package=almost-functional
Success Redirects to the homepage on the repository site for the package specified.
Shield
URL /shield/[codecov|github|gradle_plugin|jcenter|maven_central]
Method GET
Params
Param Description Required By
group The group name. jcenter, maven_central
package The package name. jcenter, maven_central
path The path to the repo, minus package name. codecov
Example http://repo-redirect.herokuapp.com/shield/maven_central?group=com.github.nwillc&package=almost-functional
Success Redirects to a shield indicating the repository type, and the latest version if it can be determined.
